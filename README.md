# BeeMall
## 概述
BeeMall是一套微服务商城系统，后台使用了spring-cloud架构。前端使用了vue。项目分为管理端和用户展示端(移动端)
## 部署启动
- 优先启动Nacos、RocketMQ、Sentinel控制台这些组件。
- 再启动Gateway
- 项目端口分配

|服务|端口|
|:---:|:---:|
|Nacos|8848|
|RocketMQ|9876|
|Sentinel Dashboard|8085|
|Gateway|8888|

  - 添加类型(类型名称)
